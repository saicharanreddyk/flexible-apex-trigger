# Flexible Apex Trigger

This is a framework that makes our Apex Trigger development more flexible.

# How to use

## Install on your org

```sh
sfdx force:package:install --package "flexible-apex-trigger@1.0.0"
sfdx force:user:permset:assign -n Logger_User
```
